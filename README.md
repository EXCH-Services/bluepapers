# EXCH Blue Papers

The EXCH Blue Papers define the conceptual architecture of the EXCH ecosystem.

They introduce the principles of Financial Navigation, LinkedID, the Global Personal Financial Statement (GPFS), and related open standards.

## Published Papers

### BP-001 — Financial Navigation
Defines the concept of Financial Navigation as a new paradigm for managing financial identity, obligations, assets, and decision-making.

### BP-002 — GPFS
Introduces the Global Personal Financial Statement (GPFS) as a universal representation of an individual's financial position.

### BP-003 — LinkedID
Defines LinkedID as the identity layer connecting individuals, assets, obligations, and financial events.
